# Security App

## Usage Guide

1. Install eas-cli: `npm install -g eas-cli`

2. APK File: Create an APK file by building the application with `npx eas build --platform android`.

3. Installation: Install the Android app using this APK file.

4. Account Creation: Register a new account. You will automatically be assigned the User role.

5. Gain Access: An Admin must manually change your role to Member.

6. View Images: Once upgraded, you can navigate to the Gallery tab to see the images, which are recorded on a Raspberry Pi Zero W and updated every 5 minutes, or checkout the Live section to view the current camera screen.
